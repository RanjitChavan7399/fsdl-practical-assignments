# Personal Portfolio Website

## Overview

This project is a responsive personal portfolio website developed as part of the Full Stack Development Lab (FSDL) practical assignments. The website showcases personal information, education, technical skills, projects, and contact details.

## Objective

To design and host a responsive personal portfolio website using modern web technologies.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap (Optional)

## Features

- Responsive Navigation Bar
- Home Section
- About Me
- Education Details
- Skills
- Projects
- Contact Information
- GitHub Profile Link
- LinkedIn Profile Link
- Responsive Design
- Footer Section
